# Scripts

This directory will contain paper-specific command-line scripts.

Planned scripts:

- `reproduce_simulation.py`: regenerate simulation results and figures.
- `prepare_wns_emoji_tsv.py`: generate the local emoji TSV from an authorized WNS corpus copy.
- `prepare_wns_lexical_tsv.py`: generate the local lexical TSV from an authorized WNS corpus copy.
- `reproduce_emoji_table.py`: regenerate the emoji top-k table and TV summaries.
- `reproduce_lexical_table.py`: regenerate the lexical sensitivity table and TV summaries.

The WNS preparation scripts are for authorized corpus users only and do not redistribute WNS-derived token-level data.
